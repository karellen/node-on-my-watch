## Using Node-On-My-Watch with Docker

A simple example is as follows:
```
$ docker run -t ghcr.io/karellen/node-on-my-watch:latest
```

## Using Node-On-My-Watch on MacOS

```
$ brew install python3.11
$ pip3.11 install nomw
$ nomw --version
```
